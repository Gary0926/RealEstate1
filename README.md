# 房價預測
## 背景
想了解大台北地區房地產價格行情，來模擬未來的買賣可能所需之價格。
## 目標
通過爬蟲(自動化)下載台北市與新北市歷年房地產價格，來降低手動下載每一年每一季檔案的繁瑣步驟。透過神經網路與隨機森林模型來對2023年的房價進行預測。
## 爬蟲網站
藉由內政部的[不動產資料供應系統](https://plvr.land.moi.gov.tw/Login_input?authfailed=true#)來下載所需資料:
![不動產資料供應系統](不動產資料供應系統.png)
## 程式碼說明
爬蟲說明請閱 **實價登錄應用--爬蟲.ipynb**。資料合併、轉換與篩選請閱 **實價登錄應用--房地產數據處理.ipynb**，模型建立與預測請閱 **實價登錄應用--房價預測.ipynb**，並特別使用colab中的GPU來加速模型運算。分析描述部分以三引號包覆表示。
